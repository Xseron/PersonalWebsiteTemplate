# My Personal Website Template

## Setup

Install the dependencies:

```bash
# Using npm
npm install

# Using pnpm (recommended)
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev
```
